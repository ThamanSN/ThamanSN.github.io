ASCEND – Autonomous UAV System

Achievement: Concept Proposal Submitted | January 2026

Team Lead. Spearheaded the design of a GPS-denied autonomous quadrotor UAV system for indoor navigation and surveillance for the ISRO Robotics Challenge (IRoC-U 2026).

View Project & Technical Details

Problem Statement: GPS signals are unavailable or unreliable in indoor environments, underground facilities, and urban canyons, making traditional UAV navigation impossible for search-and-rescue, surveillance, and inspection.

Solution: A vision-based autonomous navigation pipeline using OAK-D Pro stereo camera for depth perception and visual odometry, integrated with Pixhawk flight controller running ArduPilot firmware.

Key Features:
  • GPS-denied autonomous navigation
  • Stereo vision-based localization
  • SLAM mapping for indoor environments
  • Pixhawk + ArduPilot integration
  • Multi-sensor fusion framework

Technical Navigation & Control: Multi-sensor integration combining IMU, LiDAR/ultrasonic range sensors, and stereo vision using SLAM algorithms for precise localization. Hardware integration strategy includes MAVLink communications and real-time data processing for mission-critical operations.

Impact: Submitted innovative concept proposal to ISRO's national robotics challenge, demonstrating advanced understanding of autonomous systems, computer vision, and GPS-denied navigation techniques.

Pixhawk & ArduPilot OAK-D Pro Stereo SLAM Algorithms ROS & Python Sensor Fusion

e-Yantra Robotics Competition 2025-26

Achievement: National-Level Competition by IIT Bombay | Selected till semi finals

Overview: Developing an autonomous farm-to-collection center delivery robot capable of navigating complex paths and color-coded item retrieval.

View Project & Technical Details

Problem Statement: Agricultural logistics requires efficient automation for transporting harvested crops from fields to collection centers. Traditional manual methods are time-consuming and labor-intensive, especially for large farms.

Solution: An autonomous robot system using STM32 microcontroller with advanced line following algorithms that can navigate white/black lines, sharp curves, zigzag patterns, and dotted segments. IR array sensors enable color-coded crate identification and precise pickup/delivery operations.

Key Features:
  • Advanced line following for complex paths
  • IR sensor-based crate identification
  • STM32 embedded control architecture
  • PID-optimized motor control
  • CoppeliaSim simulation testing

Technical Navigation & Control: Real-time sensor fusion, motor control using PID optimization, and decision-making algorithms for autonomous navigation. Iterative testing and validation using CoppeliaSim robot simulation environment.

Impact: Building practical experience in autonomous robotics, embedded systems, and sensor fusion through participation in India's premier robotics competition organized by IIT Bombay.
